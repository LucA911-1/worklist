# 工作清单应用

基于腾讯云 EdgeOne 边缘函数 + KV存储 的多人协作工作管理工具。

## 功能特性

- **智能识别**：输入自然语言自动生成工作清单（支持"每天早上9点开会"、"明天下午3点提交报告"等）
- **内置日历**：月历视图，点击日期查看当日工作安排
- **工作分类**：自动区分一次性工作和重复性工作（每日/每周/每月/每年）
- **多人协作**：通过 EdgeOne KV 存储实现云端共享，多人同时编辑
- **云端同步**：自动定时同步 + 手动同步，支持离线使用
- **移动适配**：手机优先设计，支持深色模式

## 部署到腾讯云 EdgeOne

### 第一步：创建 EdgeOne Pages 项目

1. 登录 [腾讯云 EdgeOne 控制台](https://console.cloud.tencent.com/edgeone)
2. 左侧导航选择 **Pages 服务**，点击 **立即开通**
3. 点击 **创建项目**，选择 **从模板创建** 或 **连接GitHub仓库**
4. 将本项目代码推送到 GitHub 仓库，EdgeOne 会自动关联

### 第二步：配置构建参数

在项目创建页面设置：

| 参数 | 值 |
|------|-----|
| 构建命令 | `echo 'no build needed'` |
| 输出目录 | `.`（根目录） |
| 部署分支 | `main` |

### 第三步：创建 KV 存储命名空间

1. 在 EdgeOne 控制台左侧导航选择 **KV 存储**
2. 点击 **创建命名空间**
3. 填写名称：`worklist_kv`
4. 存储区域选择 **全球分布式**
5. 点击 **确定** 创建

### 第四步：绑定 KV 到边缘函数

项目部署后：

1. 进入 **边缘函数** > **函数管理**
2. 找到自动部署的 `api/tasks` 函数，点击进入详情
3. 在 **扩展服务** 模块，点击 **添加服务绑定**
4. 选择 KV 存储，命名空间选择 `worklist_kv`
5. **变量名** 填写：`WORKLIST_KV`（必须与代码中一致）
6. 保存并重新部署函数

### 第五步：获取访问地址

1. 在 Pages 项目详情页获取访问域名（如 `your-project.edgeone.app`）
2. 打开应用，进入 **设置**
3. 在 **云端地址** 填入你的域名（如 `https://your-project.edgeone.app`）
4. 填入你的名字，点击保存

### 第六步：验证多人协作

1. 在不同设备/浏览器上打开同一地址
2. 各自设置不同的用户名
3. 添加的任务会自动同步到所有设备
4. 每条任务显示创建者和修改者

## 本地开发

```bash
# 安装 EdgeOne CLI
npm install -g edgeone

# 本地开发（支持边缘函数调试）
edgeone pages dev

# 手动部署
edgeone pages deploy .
```

## 使用说明

### 智能输入示例

| 输入 | 解析结果 |
|------|---------|
| `每天早上9点开早会` | 重复性·每天·09:00·开早会 |
| `明天下午3点提交项目报告` | 一次性·明天·15:00·提交项目报告 |
| `每周一周三周五健身` | 重复性·每周一三五·健身 |
| `每月1号核对账目` | 重复性·每月1号·核对账目 |
| `下个月5号交房租` | 一次性·下月5日·交房租 |
| `紧急明天上午10点客户会议` | 一次性·明天·10:00·高优先级·客户会议 |

### 重复性工作支持

- **每天**：`每天`、`每日`
- **每周**：`每周一`、`每周一三五`、`工作日`、`周末`
- **每月**：`每月1号`、`每月15号`
- **每年**：`每年`

### 同步机制

- **自动同步**：每30秒自动同步一次（配置云端地址后生效）
- **手动同步**：点击底部 **同步** 按钮
- **离线模式**：无网络时数据保存在本地，恢复网络后自动同步
- **冲突处理**：以最后修改时间为准自动合并

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/tasks` | 获取所有任务 |
| POST | `/api/tasks` | 创建任务（支持批量） |
| PUT | `/api/tasks` | 更新任务 |
| DELETE | `/api/tasks?id=xxx` | 删除任务 |

## 技术栈

- 前端：HTML5 + CSS3 + 原生 JavaScript（无框架依赖）
- 后端：腾讯云 EdgeOne 边缘函数（Serverless）
- 存储：EdgeOne KV 存储（分布式键值数据库）
- 部署：EdgeOne Pages（全球边缘节点分发）
